## 5. 로그인 방법
1. 브라우저에서 접속
2. 이메일과 비밀번호 입력
3. 로그인 버튼 클릭

## 6. 주의사항
- 테스트 환경에서만 사용
- 실제 운영 환경에서는 반드시 비밀번호 변경 필요
- 시스템 관리자 계정은 특히 주의해서 관리

## 7. 비밀번호 변경 방법
1. 로그인 후 대시보드 접속
2. 프로필 메뉴 클릭
3. 비밀번호 변경 선택
4. 현재 비밀번호와 새 비밀번호 입력
